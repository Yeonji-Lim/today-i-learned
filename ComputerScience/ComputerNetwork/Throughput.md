# 처리량
링크내에서 성공적으로 전달된 데이터의 양
(여기서 링크는 네트워크 노드 사이 연결)

보통 얼만큼의 [트래픽](Traffic)을 처리했는지를 나타낸다.

많은 트래픽을 처리한다. == 많은 처리량을 가진다. 
## 단위
bits per second, bps

## '처리량이' 영향을 받는 것
- 커지는 트래픽
- 네트워크 장치 간의 [대역폭](Bandwidth)
- 네트워크 중간에 발생하는 에러
- 하드웨어 스펙

트래픽과 처리량을 헷갈리지 말자
트래픽이 많아짐 == 흐르는 데이터가 많아짐
처리량이 많아짐 == 처리되는 트래픽이 많아짐
