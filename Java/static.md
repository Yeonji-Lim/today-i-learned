# static
정적

객체(인스턴스)에 소속이 아니라 클래스에 고정된다.

클래스 로더가 클래스를 로딩해서 메소드 메모리 영역에 적재할 때 클래스 별로 관리된다. -> 클래스 로딩이 끝나는 즉시 바로 사용 가능

static이 붙은 것들은 다른 것들보다 먼저 컴파일 된다.
