# Java의 프로그래밍에 사용되는 3대 핵심 기술 패키지

이 3가지 기술이 어떻게 연결되어있고, 어떻게 다른지 구별하는 것이 중요하다.

## [JDK](JDK)

Java 프로그램을 구동하는 것 -> Java 프로그램 개발의 일환

그렇기 때문에 JRE를 필요로 한다. 

## [JVM](JVM)

Java 프로그램을 실행하는 자바 플랫폼의 구성요소

## [JRE](JRE)

Java 코드를 실행하기 위한 도구들로 구성된 패키지
